# @uepm/core

Core utilities for UEPM - shared code for managing Unreal Engine project files and package.json.

This package contains shared functionality used by other UEPM packages:
- UProject file reading/writing
- Package.json management
- Type definitions

## Installation

```bash
npm install @uepm/core
```

## Usage

This is typically used as a dependency by other UEPM packages, not directly by end users.
